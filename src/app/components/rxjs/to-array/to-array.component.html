toArray


toArray: A transformation operator that collects all the emitted values from the source Observable and delivers them as a single array when the source completes.
Useful when you want to wait for the entire stream to finish and then work with the collected values at once.
Example: of(1, 2, 3, 4).pipe(toArray()) â†’ emits [1, 2, 3, 4].
