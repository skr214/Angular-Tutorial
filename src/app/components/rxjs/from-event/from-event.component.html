From Event


The fromEvent operator creates an Observable that listens to a specific event (like a button click, key press, or mouse move) on a target (like a DOM element, document, or window). Each time that event happens, it emits the event object to all subscribers. This makes it easy to handle user interactions in a clean, reactive way instead of adding and removing event listeners manually.
