Take, TakeLast and TakeUntil


take: Emits only the first n values from the source Observable and then completes. Useful to limit emissions.
Example: source$.pipe(take(3)) → emits first 3 values, then completes.

takeLast: Waits for the source to complete and then emits only the last n values. Requires source completion.
Example: source$.pipe(takeLast(2)) → after completion emits the last 2 values.

takeUntil: Emits values from the source until a notifier Observable emits, then completes — useful to cancel streams (e.g., on destroy or timeout).
Example: source$.pipe(takeUntil(stop$)) or source$.pipe(takeUntil(timer(3000))).
